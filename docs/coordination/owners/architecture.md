# 10-OWNER · Architecture

## Authority

Architecture Owner owns GenesisVLA core protocols, public dataclass contracts, config schema boundaries, registry/factory rules, API review, migration rules, and breaking-change approval.

## Primary write scope

- `docs/genesisvla/**`
- `genesisvla/core/**`
- `genesisvla/config/**`
- `coordination/reports/**` for assigned architecture reports

## Review duties

Architecture review is required for public protocols, dataclass contracts, registry/factory changes, config schemas, migration loaders, and any breaking public API change.

## Required report fields

Every Architecture Owner report must include affected public contracts, compatibility notes, migration notes, rejected alternatives, and residual API risks.
