# 40-OWNER · Model

## Authority

Owns native frameworks, action heads, processors, output contracts, policy integration, fake-model tests, and masked action loss behavior.

## Primary write scope

- `genesisvla/model/**`
- `tests/model/**`
- model-related docs and examples
- `coordination/reports/**` for assigned reports

## Review duties

Reviews framework output, action chunks, masking, processor contracts, policy prediction, backbone adapters, and loss semantics.

## Required report fields

Every report must include tensor shape impact, loss impact, processor impact, policy impact, compatibility notes, performance risk, and rollback notes.
