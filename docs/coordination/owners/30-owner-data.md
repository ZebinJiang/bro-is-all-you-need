# 30-OWNER · Data

## Authority

Owns sample contracts, transforms, statistics, fixtures, mixture datasets, collators, normalization, and related tests.

## Primary write scope

- `genesisvla/dataloader/**`
- `tests/dataloader/**`
- `coordination/reports/**` for assigned reports

## Review duties

Reviews sample shape, modality names, transform order, normalization roundtrip, fixture determinism, sampling weights, and statistics.

## Required report fields

Every report must include contract impact, transform impact, fixture impact, determinism notes, copy risk, and rollback notes.
