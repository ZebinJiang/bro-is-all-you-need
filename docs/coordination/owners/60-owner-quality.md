# 60-OWNER · Quality

## Authority

Owns CI, pre-commit, pyright, test matrix, documentation gates, repository meta tests, and final quality evidence for Owner tasks.

## Primary write scope

- `.github/**`
- `.pre-commit-config.yaml`
- `pyrightconfig*.json`
- `tests/meta/**`
- quality-related docs
- `coordination/reports/**` for assigned reports

## Review duties

Reviews required command coverage, static checks, meta tests, documentation gates, validation evidence, skipped checks, and publication readiness.

## Required report fields

Every report must include commands run, pass/fail results, skipped checks, evidence path, residual risk, rollback notes, and final quality recommendation.
